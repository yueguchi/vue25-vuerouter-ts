## Build Setup

``` bash
# install 「node_modules」
1. $npm install

# 成功したら、wwwが出来上がるよ
2. $npm run build

# install 「platforms」
# config.xmlに記載されたpluginも自動でDLされます
3. $cordova platform add ios/android

# localでぱぱっとみたい時
4. $npm run dev
```
